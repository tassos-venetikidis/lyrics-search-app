# LyricsSearch

Search for lyrics for all your favorite songs via this simple UI that leverages the lyrics.ovh API to retrieve suggestions on the specified search term. The search term can be either the artist or the name of a song or both. The API returns a list of suggestions. Click on the Get Lyrics button on any suggestion to view the full lyrics of the song.

Made with:

- HTML
- CSS
- vanilla JavaScript
